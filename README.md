# DocLLM

A private, browser-based PDF viewer with RAG-powered document Q&A.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Testing

Run all tests:

```bash
npm test
```

Run unit tests only:

```bash
npm run test:unit
```

Run integration tests only:

```bash
npm run test:integration
```

Run tests with coverage:

```bash
npm run test:coverage
```
